# react-native-grid

easy to use grid layout in react-native

## Installation

```sh
npm install rn-grid-layout
```

## Usage

```js
import Grid from 'react-native-grid';

// ...

<Grid col={3} space={12}>
  //- your components
</Grid>;
```
